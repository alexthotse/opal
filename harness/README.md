# Agentic Harness

This repo ships an optional Gas Town + Beads harness for agentic work.

Entry point:

```bash
./harness/gt.sh install
./harness/gt.sh init
./harness/gt.sh rig-add
./harness/gt.sh bead "add feature X"
./harness/gt.sh sling <bead_id>
```

Defaults:
- HQ dir: `.gastown/` (gitignored)
- Rig name: `opal` (points at current repo)
